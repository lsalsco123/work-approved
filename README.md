# 환경안전 작업허가서 전산 발급 시스템

LS Alsco 환경안전 작업허가서를 웹에서 작성 → 서명 → **원본 양식 그대로** 출력(PDF)하는 Next.js 앱.

## 동작 방식

원본 엑셀 양식(`양식` 시트)을 그대로 PDF/이미지(`asset_page1.png`, `asset_page2.png`)로 추출해 **배경**으로 깔고,
입력값·체크표시(■)·공정 선택(빨간 동그라미)을 정확한 셀 좌표(`coords.json`) 위에 겹쳐 그립니다.
→ 출력물이 원본 양식과 100% 동일하게 보장됩니다. (브라우저 인쇄 → PDF)

- 좌표계: 엑셀 셀 → PDF 포인트 변환을 실측 보정(스케일 0.4044, A4 가로 2페이지).
- 체크박스 "□ ○"는 양식에 인쇄돼 있고, 선택 시 ■(업체)·●(환경안전팀 확인)를 그 위에 덮어 그림.
- 공정명은 떠 있는 도형(배치도) 위에 빨간 타원을 위치시킴.

## 실행

```bash
cd permit-app
npm install
npm run dev      # http://localhost:3000
```

- `/fill` : 작업허가서 작성 (업체/관리자 역할 전환, 라이브 미리보기, 인쇄)
- `/preview` : 검증용 샘플(신우기전) 렌더링

## 배포 (Vercel)

```bash
npm run build    # 정적 빌드, 오류 없음 확인
```
Vercel에 `permit-app` 디렉터리를 루트로 연결하면 됩니다. (서버 의존성 없음 / 정적)

## 양식이 바뀌면 (자산 재생성)

`_pipeline/` 의 스크립트로 새 양식에서 자산을 다시 만듭니다 (Python 3.13 + openpyxl, pymupdf, pillow 필요).
원본이 회사 IRM(민감도 레이블)으로 암호화된 경우, Excel에서 열어 보호 해제본(.xlsx)을 먼저 만들어야 합니다.

1. 해제본을 PowerShell+Excel COM으로 `양식` 시트만 PDF로 추출 (`_form_blank.pdf`)
2. `_extract_geometry.py` → 행/열 기하정보(`_grid.json`)
3. `_fit_affine.py` → 셀↔PDF 변환식(`_transform.json`) (PDF 격자선 실측 보정)
4. `_build_assets.py` → 배경 PNG + `asset_coords.json` (병합셀 인식)
5. `_build_marks.py` → 체크박스 □/○ 좌표(`asset_marks.json`)
6. 생성물을 `src/lib/`(coords.json, marks.json), `public/`(배경 PNG)로 복사

## 남은 작업 (다음 단계)

- **인증**: 게스트 로그인 / 관리자 로그인 (현재는 역할 토글만). 게스트 ID 발급 체계 필요.
- **서버 저장/공유**: 현재 입력은 브라우저 localStorage에만 저장. 업체 작성 → 우리 서명 워크플로를 위해 서버 저장(예: Firebase) 필요.
- 인쇄 결과 1회 실측 점검(프린터/브라우저별 여백).
